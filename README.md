# INTELLIGENT-INTRUSION-DETECTION-SYSTEM
An Intelligent Intrusion Detection System using machine learning to detect network attacks. It uses synthetic data, preprocessing, and a Random Forest model for accurate classification. The system generates alerts and is evaluated using accuracy, precision, recall, and F1-score

# Intelligent Intrusion Detection System

This project uses machine learning to detect network intrusions.

## Features
- Synthetic dataset generation
- Preprocessing & feature engineering
- Random Forest model
- Alert system
- Performance metrics

## Run
pip install -r requirements.txt  
python intelligent_intrusion_detection.py
